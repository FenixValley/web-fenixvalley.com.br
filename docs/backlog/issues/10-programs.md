## Contexto

Os programas do Fenix Valley precisam ser apresentados com clareza para atrair participantes, mentores e parceiros.

## Escopo

- Criar pagina de Programas.
- Criar paginas/sections para pre-aceleracao, aceleracao, incubacao, inovacao aberta, residencia tecnologica e programas para estudantes.
- Descrever conteudos, publico-alvo, calendario, criterios e chamada para inscricao.
- Criar fluxo de inscricao em programas.
- Preparar Demo Day e conexoes com empresas.

## Criterios de aceite

- Cada programa tem pagina com CTA especifico.
- Inscricoes ficam registradas para revisao administrativa.
- Admin consegue abrir/fechar inscricoes.
