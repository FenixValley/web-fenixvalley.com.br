## Contexto

Participantes do ecossistema precisam cadastrar perfis completos para criar conexoes confiaveis.

## Escopo

- Criar formulario de cadastro para empreendedores, startups, empresas, universidades, professores, pesquisadores, estudantes, mentores, investidores, instituicoes publicas, OSCs, organizadores de eventos e parceiros.
- Campos: nome, organizacao, area, cidade/regiao, descricao, site, redes, imagem/logo, contato, interesses, oportunidades oferecidas e contribuicao.
- Criar upload seguro de imagem/logo.
- Criar pagina publica de perfil aprovado.
- Criar botao para iniciar contato.
- Criar estados pendente, aprovado, rejeitado e arquivado.

## Criterios de aceite

- Cadastro valida campos obrigatorios e consentimento.
- Perfil aprovado fica acessivel publicamente.
- Perfil pendente nao aparece em listagens publicas.
- Admin consegue revisar solicitacoes.
