## Contexto

O Fenix Valley precisa de banco de dados, area autenticada, moderacao e painel administrativo para gerir cadastros, eventos, oportunidades, conteudos e indicadores.

## Escopo

- Escolher e configurar banco de dados e ORM.
- Modelar membros, organizacoes, startups, eventos, oportunidades, parceiros, conteudos, mentores, investidores, espacos e indicadores.
- Implementar autenticacao e perfis administrativos.
- Criar painel admin com aprovar/rejeitar cadastros.
- Criar CRUDs administrativos de eventos, oportunidades, artigos, parceiros e indicadores.
- Implementar exportacao de contatos.
- Registrar auditoria basica de alteracoes.
- Preparar politicas LGPD, consentimento e exclusao de dados.

## Criterios de aceite

- Admin geral consegue aprovar, rejeitar e editar registros.
- Permissoes impedem acesso indevido por perfil.
- Dados sensiveis tem validacao e tratamento LGPD.
- CRUDs possuem estados de loading, erro e sucesso.
